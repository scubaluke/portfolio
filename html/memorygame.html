Emotional Learning Memory Game

5th Grade level

I built and designed this memory game for SELTrove to use in their social media advertising. Students and educators can play the game as a fun way to learn about the unique lessons SELTrove offers.

I designed this project with flexbox allowing versatility and proper display on all screen types. I also implemented media queries at various screen sizes to change card size as well as font size to alow the game to be played easier on all devices.

For The text of the cards I used a JavaScript array of objects. This allows the card text to be easily changed and added to. For instance if implementing this game for a different course or age group all that needs to be done is adding a few new arrays! The game also loads fast and is very responsive using this technique.
